This is a Connect4 game, the purpose of the game is to connect four checkers in a row, horizontally, vertically or diagonally.
There are two players with their own turns. To launch this game you have to download the java files, then open a Java editor and create a new project,
then you have to create a package called "connectFour", and then drag the previously downloaded files in the package and the run the Main class. Otherwise you can go to
our repository on github at this link: https://github.com/JvckDL/progettoJava and download the project as zip and open it in a java editor, later you can run the Main class 
and start the game. There is also a JAR folder inside of the ProgettoJava folder, when you can run the game with a simple dobleclick.
